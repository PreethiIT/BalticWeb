På indenskærs isrekognoscering fredag den 1. august 2014 mellem Prins Christian Sund, Frederiksdal, Båkeruten, Nanortalik, Sydprøven, Sardloq, Julianehåbs Fjord, Skovfjord, Narsaq Sund, Bredefjord og Narsarsuaq blev følgende observeret:

23. I Bredefjord: Var der spredte isfjelde og skosser.

24. I ruten Narssaq Sund - Tasiussaq: Var der mange isfjelde og skosser.

30. I Tuno: Var der spredte isfjelde og skosser.

31. I Tunuaraq: Var der mange isfjelde og skosser.

32. I Narssaq Sund: Var der spredte isfjelde og skosser.

33. I Narsaq havn: Var der enkelte isfjelde og skosser.

34. I Skovfjord: Var der enkelte isfjelde og skosser.

35. I Tunugdliarfik: Var der spredte isfjelde og skosser. I området omkring Qoroq var der mange isfjelde og skosser.

37. I Mato løb: Var der spredte skosser samt et større isfjeld med fod der kunne passeres.

38. Ved Mato Fyret: Var der enkelte isfjelde og skosser der kunne passeres.

39. I Qaqortoq havn: Var der isfrit.

40. I Julianehåbs Fjord: Var der enkelte isfjelde og skosser.

41. I Igaliko Fjord: Var der enkelte isfjelde og skosser.

42. I ruten Qaqortoq - Niaqornatsiaq: Var der enkelte isfjelde og skosser.

43. I I kerasarssungnut: Var der enkelte isfjelde og skosser.

44. I Ikerasarssuk kujatdleq: Var der enkelte skosser.

45. Ved Niaqornatsiaq: Var der spredte isfjelde og skosser der kunne passeres.

46. I ruten Niaqornatsiaq - Sardloq: Var der enkelte isfjelde og skosser. Ud for Eqalugarrsuit var der spredte isfjelde og skosser.


48. I ruten Paggivik Fyr - Sardloq: Var der enkelte isfjelde og skosser.

49. I ruten nord og øst om Sardloq: Var der spredte isfjelde og enkelte skosser der kunne passeres.

50. Ikerasak ved Sardloq: Var der enkelte skosser.

51. I ruten syd om Sarqardlit: Var der spredte isfjelde og skosser.

52. I Umanarssup tunua: Var der spredte isfjelde og skosser.

53. I ruten syd om Imilik: Var der spredte isfjelde og skosser.

54. I ruten Imilik til Qaersup ikerasa: Var der enkelte isfjelde og skosser.

55. I Qaersup ikerasa: Var der enkelte isfjelde og skosser.

56. Nord om Qeqertarssuaq: Var der spredte isfjelde og skosser.

57. I nordlige Umanartut: Var der spredte isfjelde og skosser.

58. Syd om Qeqertarssuaq: Var der spredte isfjelde og skosser.

59. I Sydprøven Ikerasarssuk: Var der spredte isfjelde der kunne passeres af både.

60. I ruten syd om Sagdlia: Var der spredte isfjelde der kunne passeres af mindre fartøjer.

61. I ruten syd om Nunanguaq: Var der spredte isfjelde der kunne passeres.

62. I sydlige Umanartut: Var der spredte isfjelde og skosser.

63. I ruten Lichtenau Fjord: Var der enkelte isfjelde og skosser.

64. I ruten Sydprøven - Unartoq: Var der spredte isfjelde og skosser.

65. I ruten Sydprøven-Sermersoq: Var der enkelte isfjelde og skosser.

66. Nord for Arnarqat: Var der spredte isfjelde og skosser.

67. Syd for Arnarqat: Var der spredte isfjelde og skosser.

68. I ruten Unartoq – Sermersoq: Var der enkelte isfjelde og skosser.

69. Nord for Sermersoq: Var der enkelte isfjelde og spredte skosser.

70. Nordøst for Sermersoq: Var der enkelte isfjelde og skosser.

71. I ruten Kap Egede – Nalunaq: Var der spredte isfjelde og skosser.

72. Vest for Sermersoq: Var der spredte isfjelde og skosser.

73. Syd for Sermersoq: Var der spredte isfjelde og enkelte skosser.

74. Øst for Thomsens Ø: Var der enkelte skosser.

75. I Nanortalik Ikerasarssuk: Var der enkelte isfjelde og skosser der kunne passeres.

76. I Quagssup tunua: Var der spredte isfjelde og enkelte skosser.

77. I Nanortalik havn: Var der ved indsejlingen to isfjelde der kunne passeres. Havnen var fri for is.

78. I ruterne til Nanortalik omkring Ortusugssuk: Var der spredte isfjelde og enkelte skosser.

79. Ved Nanortalik i nordvest - sydøst fyrlinien: Var der spredte isfjelde og enkelte skosser.

80. Ved Nanortalik i nord - syd fyrlinien: Var der spredte isfjelde og enkelte skosser.

81. Ved Nanortalik i øst - vest fyrlinien: Var der enkelte isfjelde og skosser.

82. I Tasermiut: Var der enkelte isfjelde og skosser.

83. I båkeruten Nanortalik - Frederiksdal: Var der enkelte skosser der kunne passeres.

84. I ruten Nanortalik - Frederiksdal: Var der spredte isfjelde og skosser.

85. I ruten Frederiksdal - Augpilagtoq: Var der enkelte isfjelde og skosser. Ved Nunarssuaq og Augpilagtoq var der spredte isfjelde og skosser.

86. I ruten Augpilagtoq – Sermerunerit: Var der enkelte isfjelde og skosser.

87. I Prins Christian Sund ved det smalle sted: Var der enkelte isfjelde og skosser.
Der var et mindre bælte med spredte skosser der kunne passeres.

88. I Prins Christian Sund østlige del: Var der enkelte isfjelde og skosser. Der var områder med spredte skosser ud for bræerne.

89. I Prins Christian Sund Ø-lig ved mundingen: Var der spredte isfjelde der kunne passeres.



Næste isrekognoscering forventes onsdag den 6. august 2014.
