Welcome to ArcticWeb

An effort from Danish Maritime Authority to improve maritime safety in the Arctic region.
ArcticWeb serves as a single access point to safety related information, it provides a streamlined way to do reporting to Arctic Command and it allows for voluntary coordinated sailing thru route sharing.
If you are a ship in the area or represent an organisation that have ships in the area, you may become a user in the system via the button below.
