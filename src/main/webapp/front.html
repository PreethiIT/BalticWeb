Welcome to ArcticWeb

An effort from Danish Maritime Authority to improve maritime safety in the Arctic region. ArcticWeb serves as a single access point to safety related information, provides a streamlined way to do reporting to Arctic Command and it allows for voluntary coordinated sailing through schedule and route sharing. If you are on a vessel in the region or represent an organisation with vessels in the region, you may become a user in the system via the button below.
